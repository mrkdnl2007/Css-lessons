Position

The position property specifies the type of positioning method used for an element. It can have values like static, relative, absolute, fixed, and sticky.

div {
position: static;
top: 20px;
left: 30px;
}

Z-Index

The z-index property specifies the stack order of an element. An element with a greater z-index value will be displayed in front of elements with lower values. Note that z-index only works on positioned elements (those with a position value other than static).

.box1 {
position: relative;
z-index: 1;
}
.box2 {
position: relative;
z-index: 2;
}
.box3 {
position: relative;
z-index: 3;
}

Inline Block

Setting an element's display property to inline-block allows it to sit next to other elements while retainin block-level features like setting width, height, and margins.

.inline-box {
display: inline-block;
width: 100px;
height: 100px;
background-color: #4da6ff;
margin: 5px;
}

Align

CSS alignment can be done in different ways using properties like text-align, vertical-align, and margin auto for centering elements.

.center-text {
text-align: center;
}
.center-box {
width: 200px;
margin: 0 auto;
background-color: #b3d9ff;
padding: 10px;
}
I'm centered!

Opacity

The opacity property controls the transparency of an element. The value ranges from 0 (fully transparent) to 1 (fully opaque).

.transparent-box {
width: 100px;
height: 100px;
background-color: #0099cc;
opacity: 0.5;
}

Navigation Bar

A navigation bar helps users navigate your site. It can be styled horizontally or vertically using Flexbox or float, and enhanced with hover effects and dropdowns.

nav {
display: flex;
justify-content: space-around;
}
nav a:hover {
background-color: #0066cc;
}

Image Gallery

An image gallery is a group of images styled together. You can use CSS Grid or Flexbox to lay them out responsively.

.gallery {
display: flex;
flex-wrap: wrap;
gap: 10px;
}
.gallery img {
width: 150px;
height: auto;
border-radius: 8px;
}

Rounded Corners

Use border-radius to make corners rounded. You can apply it to any element with borders or backgrounds.

.rounded-box {
border-radius: 15px;
background-color: #cce6ff;
padding: 10px;
}
This box has rounded corners!

CSS Transitions

Transitions allow you to change property values smoothly over a duration. Commonly used for hover effects and animations.

.transition-box {
width: 100px;
height: 100px;
background-color:#3399ff;
transition: background-color 0.5s ease;
}
.transition-box:hover {
background-color: #003366;
}

CSS Animations

Animations let elements change styles over time. Use @keyframes to define the animation behavior.

@keyframes floatbox {
0% { transform: translateX(-100%); }
100% { transform: translateX(100%); }
} .animated-box {
animation: moveUpDown 2s infinite;
}
